class CpgActorController(state: ndarray[Any, dtype[float64]], num_output_neurons: int, weight_matrix: ndarray[Any, dtype[float64]], dof_ranges: ndarray[Any, dtype[float64]])

Bases: ActorController

Cpg network actor controller.

A state array that is integrated over time following the differential equation X’=WX. W is a weight matrix that is multiplied by the state array. The first num_output_neurons are the degree of freedom outputs of the controller.
