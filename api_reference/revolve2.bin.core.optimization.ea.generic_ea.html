revolve2.bin.core.optimization.ea.generic_ea.plot_ea_fitness_float module

Plot average, min, and max fitness over generations using the results of the evolutionary optimizer.

Assumes fitnesses is a floats. Installed as revolve2_plot_ea_fitness_float. See revolve2_plot_ea_fitness_float --help for usage.

main() → None

Run this file as a command line tool.

plot(database: str, process_id: int) → None

Plot fitness as described at the top of this file.

Parameters

  • database – Database where the data is stored.

  • process_id – Process id of the evolutionary process to plot.
