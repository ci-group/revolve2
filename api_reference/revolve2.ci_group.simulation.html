Create a simulation batch for multiple robots that do not interact using standard parameters.

Parameters:

  • robots – The robots to simulate.

  • terrains – The terrain to put each robot in.

  • simulation_time – See Batch class.

  • sampling_frequency – See Batch class.

  • simulation_timestep – See Batch class.

  • control_frequency – See Batch class.

Returns:

The created batch, ready for simulation.
