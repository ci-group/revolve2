revolve2.core.physics.actor.sdf package

Module contents

Functionality for converting actors to sdf.

to_sdf(physics_robot: Actor, model_name: str, position: Vector3, orientation: Quaternion) → str

Convert an actor to sdf.

Parameters

  • physics_robot – The actor to convert.

  • model_name – Name to use for the robot in sdf.

  • position – Position to use for the robot.

  • orientation – Orientation to use for the robot.

Returns

The created sdf.
