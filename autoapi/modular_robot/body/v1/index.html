to_grid() → tuple[list[list[list[modular_robot.body._module.Module | None]]], tuple[int, int, int]]

Convert the tree structure to a grid.

The distance between all modules is assumed to be one grid cell. All module angles must be multiples of 90 degrees.

The grid is indexed depth, width, height, or x, y, z, from the perspective of the core.

Returns:

The created grid with cells set to either a Module or None and a tuple representing the position of the core.

Raises:

NotImplementedError – In case a module is encountered that is not supported.
